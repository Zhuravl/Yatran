Yatran Typing Tutor

Yatran is a fun game that helps you become a better typist! It's perfect for kids who want to improve their typing skills in an enjoyable and engaging way.

The app is named after the famous "Yatran" typewriter, which was made a long time ago in Ukraine. It's like a computer keyboard, but it looks different and is a lot slower.

When you play Yatran, you need to type really fast! You have to type more than 200 letters per minute (or 40 words per minute), but don't worry if you make a few mistakes. You can choose different levels to make the game easier or harder, so it's always challenging and exciting.

Download the latest version of Yatran and start typing!
